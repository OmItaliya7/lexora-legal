# ⚖️ Lexora Legal – Modern Law Firm Website

Lexora Legal is a modern and fully responsive law firm website built using the MERN stack ecosystem. The platform allows users to explore legal services, attorney profiles, practice areas, request consultations, and securely manage their accounts.

## 🚀 Live Demo

Frontend: https://lexora-legall.vercel.app/

Backend API: https://your-backend-url.com

---

## ✨ Features

### Public Features

* Responsive design for Mobile, Tablet and Desktop
* Dynamic Practice Area pages
* Dynamic Attorney Profile pages
* Testimonials Carousel
* FAQ Accordion Section
* Pricing Plans
* Trusted Companies Marquee
* Contact Form Submission
* Free Consultation Request Form
* SEO Optimized Pages
* XML Sitemap Support
* Robots.txt Support

### Authentication Features

* Email & Password Registration
* Email & Password Login
* Google OAuth Authentication
* Forgot Password Functionality
* Reset Password via Email Link
* Protected User Session Handling
